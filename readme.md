# Aplikasi Raport Gold

# Instalasi
1. Clone repositori ini.
2. Buka terminal dan navigasi ke direktori proyek.
3. Jalankan perintah `npm install` untuk menginstal semua dependensi.
4. jalankan perintah 'sequelize db:seed:all' untuk menjalankan fungsi login.

# Alur :
1. Admin
    1. register admin hanya dapat di jalankan menu 'sequelize db:seed:all' sebab aplikasi ini digunakan oleh pihak internal
    2. admin bisa menginput data berbagai user dan bisa mengecek semua menu yang ada di dalam aplikasi raport ini.
    3. banyak menu yang belum dapat terselesaikan sebab alur project yang dikerjakan cukup signifikan.
